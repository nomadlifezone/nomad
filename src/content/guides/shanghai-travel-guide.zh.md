# 上海完全旅游攻略

上海，中国最国际化的大都市，融合了东西方文化。从百年外滩到现代摩天大楼，从传统弄堂到时尚商圈，本攻略带你玩转魔都。

## 🗺️ 必游景点

### 1. 外滩 ⭐⭐⭐⭐⭐
**特点**：上海标志性景观，万国建筑博览群

**推荐游览**：
- 最佳时间：傍晚5-8点（看日落和夜景）
- 观景地点：外滩观光平台、和平饭店顶楼
- 游览时长：2-3小时

**门票**：免费

**交通**：地铁2号线/10号线 南京东路站

### 2. 东方明珠/上海中心 ⭐⭐⭐⭐⭐
**东方明珠**：
- 门票：¥160-220（不同观景层）
- 特色：上海经典地标
- 高度：468米

**上海中心大厦**：
- 门票：¥180
- 特色：中国第一高楼
- 高度：632米（观光层546米）

**推荐**：选择上海中心，景观更震撼

### 3. 豫园 + 城隍庙 ⭐⭐⭐⭐
**豫园**：
- 门票：¥40
- 特色：明代古典园林
- 游览时长：1.5小时

**城隍庙**：
- 门票：¥10
- 特色：道教庙宇，周边小吃街

**美食**：南翔小笼包、梨膏糖、五香豆

### 4. 田子坊 ⭐⭐⭐⭐
**特点**：石库门里弄改造的艺术街区

**推荐活动**：
- 逛特色小店和画廊
- 品尝创意咖啡和小吃
- 拍照打卡
- 游览时长：2-3小时

**门票**：免费
**交通**：地铁9号线 打浦桥站

### 5. 上海迪士尼乐园 ⭐⭐⭐⭐⭐
**门票**：¥399-799（淡旺季不同）
**游览时长**：全天（10小时以上）
**必玩项目**：
- 创极速光轮 ⚡
- 飞越地平线 🌍
- 加勒比海盗 ⚓
- 七个小矮人矿山车 ⛏️

**省钱技巧**：
- 工作日票价更低
- 早鸟票提前购买
- 购买早享卡提前入园

### 6. 南京路步行街 ⭐⭐⭐⭐
- 中国最著名的商业街
- 各大品牌云集
- 游览时长：2-3小时
- 夜景更美

### 7. 新天地 ⭐⭐⭐⭐
- 石库门建筑风格
- 高端餐饮和酒吧
- 适合晚上去
- 门票：免费

### 8. 朱家角古镇 ⭐⭐⭐⭐
- 江南水乡风貌
- 距市区1小时车程
- 门票：免费（部分景点收费）
- 游览时长：半天

## 📅 推荐行程

### 3天经典路线

**Day 1：外滩 + 南京路**
- 上午：豫园、城隍庙（3小时）
- 中午：城隍庙小吃
- 下午：南京路步行街购物（2小时）
- 傍晚：外滩看夜景（2小时）
- 晚餐：外滩餐厅

**Day 2：浦东 + 现代上海**
- 上午：上海中心观景台（2小时）
- 中午：陆家嘴商圈午餐
- 下午：上海博物馆或科技馆（3小时）
- 晚上：东方明珠周边夜景

**Day 3：文艺上海**
- 上午：田子坊（2小时）
- 中午：田子坊周边餐厅
- 下午：新天地（2小时）
- 晚上：淮海路购物

### 5天深度路线
在3天基础上增加：

**Day 4：上海迪士尼**
- 全天游玩迪士尼乐园
- 晚上观看烟花秀

**Day 5：古镇 + 休闲**
- 上午：朱家角古镇（半天）
- 下午：返回市区，静安寺商圈
- 晚上：陕西南路美食街

## 🚇 交通指南

### 到达上海
**浦东机场**：
- 磁悬浮 → 地铁2号线（¥50 + 地铁票）
- 机场大巴（¥30，到市区各地）
- 出租车（¥150-200，1小时）

**虹桥机场/火车站**：
- 地铁2号线/10号线直达
- 出租车到市区¥50-80

### 市内交通
**地铁** ⭐⭐⭐⭐⭐（推荐）
- 线路覆盖全面
- 单程¥3-10
- 使用支付宝/微信乘车码

**出租车/网约车**
- 起步价¥14（3公里）
- 市区打车通常¥20-50

**共享单车**
- 哈啰、美团单车
- 每小时¥1.5-2

## ⚠️ 注意事项

### 安全提示
- 外滩人多，注意防扒手
- 不要相信路边拉客的"茶馆"、"酒吧"
- 夜晚避免偏僻小巷

### 最佳旅游时间
- **春季（3-5月）**：天气舒适 ⭐⭐⭐⭐⭐
- **秋季（9-11月）**：气候宜人 ⭐⭐⭐⭐⭐
- **夏季（6-8月）**：炎热多雨 ⭐⭐⭐
- **冬季（12-2月）**：寒冷但游客少 ⭐⭐⭐

### 实用建议
1. 下载上海地铁App和大众点评
2. 提前预订热门餐厅
3. 迪士尼需至少提前1天购票
4. 准备少量现金（大部分地方支持移动支付）
5. 外滩周末人超多，建议工作日去
