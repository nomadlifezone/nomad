# Complete DiDi Ride-Hailing Guide

DiDi Chuxing is China's largest ride-hailing platform, similar to Uber. This guide teaches you how to use DiDi for convenient travel in China.

## 📱 Download and Registration

### Download App

**DiDi Chuxing (DiDi)**
- iOS: Search "DiDi" in App Store
- Android: Search "DiDi" in app stores

**International Version**:
- DiDi has international app (DiDi Rider)
- Supports English interface
- Functions basically same as Chinese version

### Registration Process

1. **Open App**
2. **Choose Registration Method**:
   - Phone number registration (recommended)
   - WeChat login
   - Apple ID login

3. **Phone Number Registration Steps**:
   - Select country/region code (e.g., +86 China, +1 US)
   - Enter phone number
   - Receive SMS verification code
   - Enter verification code
   - Set nickname

4. **Bind Payment Method** (can skip, must bind before first ride):
   - Alipay
   - WeChat Pay
   - Bank card (Chinese bank card)
   - International credit card (Visa/Mastercard/Amex)

**Recommendation**: "Express" is sufficient for daily use, clean, comfortable, reasonably priced.

### Special Vehicle Types

- **Budget Carpool**: Most economical, need wait for other passengers
- **Taxi**: Traditional taxis, fixed prices
- **Designated Driver**: Driver drives your car
- **Hitch Ride**: Private cars giving rides (need advance booking)

## 📍 How to Book a Ride

### Basic Booking Process

**Step 1: Set Pickup and Destination**
1. Open DiDi App
2. **Pickup Location** (auto-locates current position):
   - Can manually enter address
   - Or drag pin on map

3. **Destination**:
   - Tap "Where to?"
   - Enter destination (supports Chinese/Pinyin/landmarks)
   - Select specific location

**Step 2: Select Vehicle Type**
- Tap "Express," "Premier," etc. to choose
- System shows estimated price and arrival time

**Step 3: Confirm Booking**
- Review estimated price
- Tap "Call Express"
- Wait for driver to accept (usually 10-30 seconds)

**Step 4: Wait for Driver**
- After driver accepts, you can see:
  - Driver name, photo
  - License plate
  - Car model
  - Driver rating
  - Estimated arrival time
- Track driver's real-time location on map

**Step 5: Board**
1. Verify license plate and car model
2. Can ask driver's name to confirm
3. Trip starts automatically after boarding

**Step 6: During Trip**
- See real-time route on map
- Can chat or call driver through app
- Can adjust drop-off location before arriving

**Step 7: Arrive and Pay**
1. After arrival, driver taps "Arrive"
2. System auto-calculates fare
3. Auto-deducts from bound payment method
4. Can rate driver after successful payment

## 💰 Pricing and Payment

### Price Components

- **Base fare**: ¥10-14 (varies by city)
- **Distance fee**: ¥1.5-2.5/km
- **Time fee**: ¥0.3-0.5/minute
- **Long distance fee**: Extra charge over 15 km
- **Night fee**: 23:00-5:00, add 20-30%
- **Peak fee**: May surge during rush hours

### Estimated Price Examples (Beijing)

- **3 km**: ¥12-15
- **10 km**: ¥30-40
- **20 km**: ¥60-80

### Payment Methods

1. **Alipay** ⭐⭐⭐⭐⭐ (Recommended)
2. **WeChat Pay** ⭐⭐⭐⭐⭐
3. **Bank Card** (Chinese)
4. **International Credit Card** (Visa/Mastercard)
5. **DiDi Coupons/Balance**

**Set Default Payment**:
- Me → Wallet → Payment Methods
- Select default payment

### Coupons

- New users get coupons
- Frequent promotions
- Check in "Wallet" - "Coupons"

## 💡 Useful Features

### 1. Schedule Ride

For advance planning (e.g., catching a flight):
1. Select "Schedule"
2. Set time (2 hours - 4 days ahead)
3. Driver arrives in advance

**Fee**: Booking fee ¥5-10

### 2. Airport Pickup/Drop-off

- Dedicated airport transfer service
- Driver waits at exit
- Free waiting time included

### 3. Share Trip

Share real-time location with family/friends:
- Tap "Share Trip" during ride
- Send to WeChat contacts
- They can track your location in real-time

### 4. Saved Addresses

Save frequently used addresses (home/work):
- Personal Center → Saved Addresses
- Add address with label
- Quick select when booking

### 5. Contact Driver

- **In-app chat**: Send text messages
- **Call**: Tap phone icon by driver photo
- Virtual number protection, neither party sees real number

## ⚠️ Safety Tips

### Before Boarding

- ✅ Verify license plate, car model, driver info
- ✅ Take photo of license plate and send to friend
- ✅ Don't board cars that haven't accepted your order
- ❌ Don't pay cash in advance

### During Trip

- ✅ Fasten seatbelt
- ✅ Share trip with family/friends
- ✅ Check if driver follows navigation route
- ❌ Don't fall asleep (when alone)
- ❌ Don't reveal personal info to driver

### Emergency Situations

- **Emergency contact**: Set emergency contacts (in app)
- **Call police**: Dial 110
- **Complaint**: Order details → Complaint/Suggestions
- **Customer service**: Contact via app

### Female Passenger Safety

- Try not to ride alone at night
- Turn on "Share Trip"
- Sit in back seat
- Stay alert

### Rating and Complaints

- Rate driver after each trip (5 stars)
- Report problems promptly
- Contact customer service for serious issues

## 🌟 Usage Tips

### 1. Increase Acceptance Rate

- Choose pickup points with easy parking
- Consider adding tip during peak hours
- Be patient, don't cancel frequently

### 2. Save Money

- Use coupons
- Choose "Carpool" mode
- Avoid peak hours
- Watch for platform promotions

### 3. Quick Booking

- Save frequently used addresses
- Use voice input for destination
- Plan routes in advance

### 4. Communication Tips

**If you don't speak Chinese**:
- Use app's "I don't speak Chinese" feature
- Prepare common phrases:
  - "Please go to this address" (point at phone)
  - "Thank you" (xiè xiè)
  - "Stop" (tíng chē)
- Use translation app

### 5. Avoid Disputes

- Confirm destination before boarding
- Don't eat in car
- Keep car clean
- Remember to take belongings when getting off

## 🎯 Common Questions

### Q1: Can foreigners use DiDi?

**A**: Yes! DiDi supports:
- Foreign phone number registration
- International credit card payment
- English interface

### Q2: What if no driver accepts?

**A**: Try:
- Wait a bit and try again
- Switch vehicle type (e.g., "Premier")
- Adjust pickup location
- Add tip
- Switch to "Taxi" mode

### Q3: Driver calls but I don't understand?

**A**:
- Use in-app chat (auto-translation)
- Send pickup location screenshot
- Or don't answer, driver will follow navigation

### Q4: Want to change destination during trip?

**A**:
- Communicate with driver
- Or modify destination in app
- Fare recalculated

### Q5: Left something in car?

**A**:
1. Contact driver immediately (call from order page)
2. Or contact customer service for help
3. Provide item description and license plate

### Q6: What's difference between DiDi and Uber?

**A**:
- DiDi acquired Uber China business
- Usage basically the same
- DiDi more prevalent in China
